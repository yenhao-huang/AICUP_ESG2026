# ★ 最重要:投票分歧(soft vote 真正在哪幾筆做決定)

> **>90% 的列全員(或近全員)一致 → 軟/硬投票結果都一樣。**
> **ensemble / 調 threshold 的勝負手,只在「強分歧(3:2 / 2:3)」那一小撮列。**
